# 广度优先遍历

逐层遍历, 创建队列, 从根节点开始遍历, 将子节点放入队列尾部, 从队列开头取出节点，循环至队列无节点

```bash
     0       [0] -> [1, 2]                       # 0 节点出队, 1 2 子节点入队尾
  1     2    [1, 2] -> [2, 3, 4] -> [3, 4, 5, 6] # 1 节点出队, 3 4 子节点入队尾, 2 节点出队, 5 6 子节点入队尾
3  4  5  6   [3, 4, 5, 6] -> [4, 5, 6] -> [5, 6] -> [6] -> []

[0] -> [1, 2] -> [2, 3, 4] -> [3, 4, 5, 6] -> [4, 5, 6] -> [5, 6] -> [6] -> [] 
```
